Рахунки для сплати платежів

1. Реквізити рахунків для сплати податків, зборів та платежів до державного та місцевих бюджетів по Волинській області

2. Реквізити рахунків для сплати єдиного внеску на загальнообов’язкове державне соціальне страхування по Волинській області

3. Реквізити рахунків за надходженнями до місцевих бюджетів Автономної Республіки Крим та міста Севастополя для сплати боргів з місцевих податків та зборів розміщені на веб-порталі Головного управління ДПС у Херсонській області, в рубриці «Рахунки для сплати платежів» https://kherson.tax.gov.ua/rahunki-dlya-splati-platejiv.
